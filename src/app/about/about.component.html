TinyGU.de   &   goethe.link Der URL Shortener der Goethe-Universität Frankfurt am Main

TinyGU ist eine Eignentwicklung der Goethe Universität durch PhysikOnline. Mit dieser Seite wird ein URL Shortener bereitgestellt der ausschließlich auf Webseiten die von der Goethe Universität betrieben werden verweist. Dadurch ist bei Nutzung dieses Dienstes gewährleistet, dass man nur auf Seiten der Goethe Universität weitergeleitet wird.

Entwickelt wurde TinyGU als OpenSource Projekt von Physikonline. Der source code ist auf GitHub einsehbar. Zum Mitmachen wird eingeladen!

Das Frontend wurde mit Angular entwickelt und findet sich hier: https://github.com/PhysikOnline-FFM/tinyGU/
Das Backend wurde mit .NET core 2 entwickelt und findet sich hier: https://github.com/PhysikOnline-FFM/tinygu_backend
